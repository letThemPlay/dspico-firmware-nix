# [DSRomEncryptor](https://github.com/Gericom/DSRomEncryptor)

Requires files to be supplied by the user to run correctly; place the following files in this folder. Flake doesn't currently support using the ntrBlowfish.bin and twlBlowfish.bin files. 

- ds arm7 bois dump named `biosnds7.rom` (SHA1: `24F67BDEA115A2C847C8813A262502EE1607B7DF`)
- dsi arm7i bios dump named `biosdsi7.rom` (SHA1: `A3AA751EB6BDAAF8A827BA9E03576A6F1AB0F547` for an incomplete dump or `C7C7570BFE51C3C7C5DA3B01331B94E7E7CB4F53` for a complete dump)

# [dspico-wrfuxxed](https://github.com/LNH-team/dspico-wrfuxxed)

If building the wrfuxxed exploit version of the firmware an additional file will be required to be placed in this directory.

- `WRFU Tester` named `dsimode.nds` version 0.60 (SHA-1: 2d65fb7a0c62a4f08954b98c95f42b804fccfd26).
